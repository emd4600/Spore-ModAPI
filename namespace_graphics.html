Graphics namespace

All the graphics system: rendering, shaders, models, lighting, textures,...

Contents

  • Reference
    • Namespaces
    • Classes
    • Enums
    • Typedefs
    • Functions

This namespace includes classes related to graphics rendering in the game. Some things included are models, textures and materials.

This namespace is deeply related with the RenderWare namespace; elements in Graphics are high-level representations, easy to use, whereas RenderWare elements are low-level representations which are not needed to modify in most cases.

  • Graphics::Model represents a 3D model made of triangulated meshes. It uses objects like RenderWare::VertexBuffer, RenderWare::IndexBuffer, RenderWare::Animations,...
  • Graphics::Texture represents a game texture, which can be 2D or 3D (cube textures). It uses RenderWare::Raster.
  • Graphics::Material contains the information of how a model mesh must be painted, that is, which shaders and textures it uses. It uses RenderWare::CompiledState
