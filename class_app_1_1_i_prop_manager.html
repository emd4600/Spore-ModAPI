App::IPropManager class
#include <Spore/App/IPropManager.h>

A manager that stores all the property lists in the game, used for most configurations.

Contents

  • Reference
    • Derived classes
    • Public static functions
    • Public functions

Property lists (.prop files) are simple instances of the class PropertyList that contains properties. This is used for configuring most objects and things in the game.

A property manager can also contain definitions for properties, which are stored in the Properties.txt and AppProperties.txt files.
